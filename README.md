# MotoMotion
a motorcycle tuning utility

## Run the modular app (`adv-trace`)

### Prerequisites
- Node.js 18+
- npm 9+

### Start locally
```bash
cd adv-trace
npm install
npm run dev
```

Open the local URL printed by Vite (typically `http://localhost:5173`).

### Production build
```bash
cd adv-trace
npm run build
npm run preview
```
